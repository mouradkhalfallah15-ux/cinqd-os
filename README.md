### 1. 🏗️ المراكب الأساسية (Core Modules)
- **مركب الصناعة (`/factory`):** إدارة الروسات، أوامر الإنتاج، ومراقبة الجودة (pH/Density) مع خصم آلي للمخزن.

- **مركب المالية (`/finance`):** محاسبة تونسية احترافية، احتساب الـ TVA والـ RS، مع "وحدة المناورة الجبائية" (نظام الـ 40%).
